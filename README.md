# QuickCommerce Sales Analytics Dashboard

This project is an end-to-end Power BI dashboard created using a simulated QuickCommerce dataset for the NCR region. The main objective of this project is to analyze sales performance, customer behaviour, product performance, and area-wise business trends through interactive visualizations.

The dataset contains 50,000 sales records and follows a Star Schema data model consisting of one fact table and multiple dimension tables. Data preparation and transformation were carried out using Power Query, while DAX was used to create KPIs and business calculations.

Tools Used

• Microsoft Power BI
• Microsoft Excel
• Power Query
• DAX
• Star Schema Data Modeling

## Dashboard Includes

• Executive Dashboard
• Area Analysis
• Product Analysis
• Customer Analysis

## Key Metrics

• Total Revenue
• Total Profit
• Total Orders
• Average Order Value
• Revenue Growth
• Profit Growth
• Orders Growth

## Project Highlights

• Cleaned and transformed the raw data using Power Query.
• Built a Star Schema data model for efficient reporting.
• Created interactive dashboards with slicers and filters.
• Developed DAX measures for business KPIs.
• Designed dashboards to support business decision-making.

## Dataset Information

• Domain: QuickCommerce (E-commerce)
• Region: NCR
• Total Records: 50,000
• Data Source: Microsoft Excel
